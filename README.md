# Rota's 12 Problems — Unranking Algorithms

This project is based on the paper [*Lexicographic Generators for Rota's Twelve Problems*](https://drops.dagstuhl.de/entities/document/10.4230/LIPIcs.AofA.2024.17) by Amaury Curiel and Antoine Genitrini (AofA2024), which introduced the first lexicographic unranking algorithms for the 12 fundamental combinatorial structures identified by Gian-Carlo Rota in the 1960s. It includes:

- An implementation of the unranking algorithms for Rota's 12 problems in TypeScript, with a full test suite
- A web application to run and visualise the algorithms interactively — see [`web/README.md`](./web/README.md)

It is intended as a contribution to [SymPy](https://docs.sympy.org/latest/modules/combinatorics/index.html), the Python library for symbolic mathematics.
